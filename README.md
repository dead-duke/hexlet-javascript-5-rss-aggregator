## Overview

The RSS Aggregator is a web application designed to collect and display content from various RSS feeds. Enjoy a streamlined reading experience with automatic updates, read status tracking, and multi-language support in a modern, responsive interface.

## Requirements

- Node.js 18+
- npm 9+
- Make

## Technology Stack

### Core Dependencies

- [axios](https://github.com/axios/axios) - HTTP requests
- [onChange](https://github.com/Qard/onchange) - reactive state
- [i18next](https://www.i18next.com) - internationalization
- [yup](https://github.com/jquense/yup) - validation

### Development Tools

- [Vite](https://vite.dev) - fast build tool and development server
- [ESLint](https://eslint.org) - code linting and quality

### Deployment & CI/CD

- [Vercel](https://vercel.com) - deployment and hosting
- [Sonar](https://www.sonarsource.com) - code quality analysis
- [GitHub Actions](https://github.com/dead-duke/hexlet-javascript-5-rss-aggregator/actions) - continuous integration

### UI Framework

- [Bootstrap 5](https://getbootstrap.com) - responsive CSS framework

## Installation and Usage

```bash
# Clone repository
git clone git@github.com:dead-duke/hexlet-javascript-5-rss-aggregator.git

# Install dependencies
make install

# Start development server
make develop
```
